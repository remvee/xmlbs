Installation instructions for XMLBS

Requirements

System requirements

Java Development Kit (tested with version 1.3)
http://java.sun.com/j2se/
Ant (tested with version 1.5.1)
Apache Ant is a Java-based build tool. http://jakarta.apache.org/ant/

Offline requirements

These will be downloaded automatically when running the build process.

Jakarta Regexp (tested with version 1.2)
A Regular Expressions library for Java. http://jakarta.apache.org/regexp/
This is a runtime dependency.
Java GetOpt (tested with version 1.0.9)
A getopt(3) alike library for Java.
This is only a runtime dependencies for the commandline classes.
JUnit (tested with version 3.8.1)
Java unit testing framework. http://www.junit.org/
Only needed to compile and run the unit tests.

For more download details read depend.xml.

Building

Extract the downloaded zipfile, enter the toplevel directory and enter: "ant -projecthelp". This will list all useful build targets.

Most useful:

jar
This target will create a jar file containing all the XMLBS classes. Please note the runtime dependencies are not included in this jar file.
apidoc
Generate JavaDoc HTML documentation for developers using this code.

Running commandline tools

At this time XMLBS includes only one simple commandline tool for translating HTML to XHTML. Make sure the regexp, getopt and xmlbs jars are in your classpath and run: "java xmlbs.XHTMLlizer -?" for more instructions.
